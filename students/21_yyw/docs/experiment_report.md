# 线性回归参数估计方法比较实验报告

## 一、实验目的

1. 掌握线性回归的最小二乘估计方法
2. 比较三种不同工具（公式法、sklearn、statsmodels）的估计结果
3. 验证三种方法的一致性

## 二、实验设置

### 数据生成过程 (DGP)

- 真实模型：$y = \beta_0 + \beta_1 x + \varepsilon$
- 真实参数：$\beta_0 = 1$，$\beta_1 = 2$
- 样本数量：$n = 100$
- 误差项：$\varepsilon \sim N(0, 1)$
- 特征变量：$x \sim U(0, 10)$

### 三种估计方法

| 方法 | 说明 | 实现方式 |
|------|------|----------|
| **公式法** | 直接使用最小二乘公式 | $\hat{\beta}_1 = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2}$，$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$ |
| **sklearn** | 机器学习库 | `LinearRegression().fit(X, y)` |
| **statsmodels** | 统计推断库 | `sm.OLS(y, X).fit()` |

## 四、结果分析

### 4.1 一致性结论

三种方法的估计结果**完全相同**（在浮点精度范围内）。这说明：

- 三种方法都基于相同的最小二乘估计原理
- 不同库的实现虽然 API 不同，但数学本质一致
- 最小二乘估计具有唯一解析解

### 4.2 偏差分析

- $\beta_0$ 偏差：**正偏差**（估计值 > 真实值）
- $\beta_1$ 偏差：**负偏差**（估计值 < 真实值）

**原因**：由于随机误差 $\varepsilon$ 的存在，单次抽样的估计值不会恰好等于真实值。但随着样本量增大，偏差会趋于 0（无偏性）。

### 4.3 理论依据

最小二乘估计量 $\hat{\beta}$ 满足：
- **无偏性**：$E[\hat{\beta}] = \beta$
- **一致性**：$\hat{\beta} \xrightarrow{p} \beta$（当 $n \to \infty$）
- **有效性**：在所有线性无偏估计中方差最小

## 五、实验结论

1. ✅ **一致性验证通过**：公式法、sklearn、statsmodels 三种方法得到的参数估计结果完全一致

2. ✅ **无偏性验证**：虽然单次估计存在偏差，但偏差值较小，符合理论预期

3. ✅ **工具选择自由**：在实际应用中，可以任选一种方法：
   - 若需要**统计推断**（p值、置信区间），推荐 statsmodels
   - 若需要**预测**和**机器学习集成**，推荐 sklearn
   - 若需要**教学演示**和**理解原理**，推荐公式法
