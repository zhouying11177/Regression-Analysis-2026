# 报告：简单线性回归的随机模拟与多方法验证

## 1. 实验背景与数据生成

本实验通过蒙特卡洛方法模拟线性回归过程。设定真实模型参数为 $\beta_0 = 1$ (截距) 和 $\beta_1 = 2$ (斜率)，并加入标准正态分布噪声 $\epsilon \sim N(0, 1)$。共生成 $N=100$ 组观测样本。

## 2. 参数估计方法对比

实验分别采用了**公式手动推导**、**`sklearn` 机器学习库**以及 **`statsmodels` 统计库**三种方式进行求解。结果如下表所示：

| **估计方法**              | **β^0 (Intercept)** | **β^1 (Slope)** | **备注**             |
| ------------------------- | ------------------- | --------------- | -------------------- |
| **真实值 (Ground Truth)** | 1.000000            | 2.000000        | 理论设定的原始参数   |
| **公式手动计算**          | 1.215096            | 1.954023        | 基于 LSE 正规方程    |
| **Scikit-Learn**          | 1.215096            | 1.954023        | 常用工业级预测模型   |
| **Statsmodels**           | 1.215096            | 1.954023        | 侧重于统计推断与检验 |

> **结论**：三种方法的数值结果在小数点后 14 位保持一致，验证了算法实现的等价性。

------

## 3. 误差与方差分析

### 3.1 估计偏误 (Bias)

偏误反映了估计值与真实值之间的距离：

- **Intercept Bias ($\hat{\beta}_0 - \beta_0$):** $0.2151$
- **Slope Bias ($\hat{\beta}_1 - \beta_1$):** $-0.0460$

### 3.2 估计精度

- **$\hat{\beta}_1$ 的方差:** $0.000939$

  该方差值较小，说明在当前样本量（N=100）下，模型对斜率的估计非常稳定。

------

## 4. 假设检验与显著性

利用 `statsmodels` 进行了回归系数的显著性检验及整体模型的方差分析（ANOVA）。

### 4.1 显著性检验 (T-test)

- **P-value ($\beta_1$):** $1.35 \times 10^{-81}$

  由于 P 值远小于常用的显著性水平（如 0.05），我们**拒绝原假设** $H_0: \beta_1 = 0$。这表明自变量 $x$ 对因变量 $y$ 具有极显著的线性影响。

### 4.2 方差分析表 (ANOVA)

方差分析通过分解总离差平方和（SST）来评估模型的解释能力：

| **来源**            | **自由度 (df)** | **平方和 (Sum Sq)** | **F 统计量** | **Prob (F)**           |
| ------------------- | --------------- | ------------------- | ------------ | ---------------------- |
| **回归 (x)**        | 1.0             | 3345.318            | 4064.56      | $1.35 \times 10^{-81}$ |
| **残差 (Residual)** | 98.0            | 80.658              | -            | -                      |

**F 检验结果**：$F(1, 98) = 4064.56$，进一步证实了模型整体的有效性。

------

## 5. 实验总结

通过本次模拟实验，我们成功验证了：

1. **算法一致性**：无论是底层公式还是封装库，其 OLS 实现逻辑高度统一。
2. **估计有效性**：在噪声干扰下，估计值虽有微小偏误，但始终围绕真实值波动。
3. **统计显著性**：极小的 P 值和高 F 值说明线性关系在统计学上是确定无疑的。
