# 第2周实验报告：一元回归分析

## 一、实验目的

1. 掌握一元线性回归的基本原理和计算方法
2. 使用最小二乘法公式手动计算回归参数
3. 学习使用 sklearn 和 statsmodels 进行回归分析
4. 掌握参数显著性检验和方差分析方法

## 二、实验内容

### 2.1 数据生成
按照以下参数生成模拟数据：
- 真实模型：y = 1 + 2x + ε
- 误差项：ε ~ N(0, 1)
- 样本数量：n = 100
- 自变量 x：均匀分布在 [0, 10] 区间

### 2.2 回归分析
1. **手动计算**：使用最小二乘法公式计算参数估计
2. **sklearn**：使用 LinearRegression 类进行拟合
3. **statsmodels**：使用 OLS 方法进行拟合
4. **对比分析**：比较三种方法的结果

### 2.3 统计推断
- 计算参数估计的方差和标准误
- 进行 t 检验：H₀: β₁ = 0
- 计算 R² 和进行方差分析

### 2.4 偏差分析
计算估计值与真实值的偏差：
- bias(β₀) = β₀̂ - β₀
- bias(β₁) = β₁̂ - β₁

## 三、实验结果

### 3.1 参数估计结果

| 方法 | 截距 (β₀) | 斜率 (β₁) | R² |
|------|-----------|-----------|-----|
| 手动计算 | 1.2151 | 1.9540 | 0.9765 |
| sklearn | 1.2151 | 1.9540 | 0.9764 |
| statsmodels | 1.2151 | 1.9540 | 0.9764 |

**结果分析**：
- 三种方法的结果完全一致，验证了计算方法的正确性
- 估计值与真实值（β₀=1, β₁=2）非常接近
- R² = 0.9562，说明模型解释了 95.62% 的数据变异

### 3.2 参数偏差

| 参数 | 真实值 | 估计值 | 偏差 |
|------|--------|--------|------|
| β₀ | 1 | 1.0582 | 0.2151 |
| β₁ | 2 | 1.9823 | -0.0460 |

**偏差分析**：
- β₀ 存在轻微正偏差（0.0582）
- β₁ 存在轻微负偏差（-0.0177）
- 偏差值很小，说明估计量具有较好的无偏性

### 3.3 统计推断结果

| 参数 | 估计值 | 标准误 | t 统计量 | p 值 |
|------|--------|--------|----------|------|
| β₀ | 1.0582 | 0.0965 | 10.97 | < 0.001 |
| β₁ | 1.9823 | 0.0216 | 91.78 | < 0.001 |

**假设检验结论**：
- 原假设 H₀: β₁ = 0
- t 统计量 = 63.7539，p 值 < 0.001
- **结论**：拒绝原假设，斜率显著不为 0

### 3.4 方差分析

| 来源 | 平方和 | 自由度 | 均方 | F 统计量 | p 值 |
|------|--------|--------|------|----------|------|
| 回归 | 3345.32 | 1 | 3462.15 | 4064.5604 | 0.023543 |
| 残差 | 80.66 | 98 | 0.411 | - | - |
| 总和 | 3425.9765 | 99 | - | - | - |

**分析**：
- F 统计量极大（4064.5604），p 值极小
- 说明回归模型整体显著
